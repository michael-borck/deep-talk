# Keyboard Shortcuts

Press **`Cmd/Ctrl + ?`** anywhere in DeepTalk to open the interactive shortcuts dialog with a search box. This page is the same list in reference form.

On macOS, use **`Cmd`**. On Windows and Linux, use **`Ctrl`**.

## Navigation

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl + 1` | Dashboard |
| `Cmd/Ctrl + 2` | Library |
| `Cmd/Ctrl + 3` | Projects |
| `Cmd/Ctrl + 4` | Search & Filter |
| `Cmd/Ctrl + 5` | Settings |
| `Cmd/Ctrl + ,` | Open Settings |
| `F11` | Toggle fullscreen |
| `Esc` | Close modal / cancel |

## Tips

- **Most shortcuts are context-aware.** `Space` toggles play/pause only when the audio player has focus, so it won't fight text input.
- **The shortcuts dialog has a search box.** Press `Cmd/Ctrl + ?` and start typing to filter down to what you need.
- **Settings has no Save button.** Everything commits as you change it вЂ” no keyboard shortcut needed.
